  2. Trình bày cú pháp xác định vùng chọn với element(div,h1,p,..), class,ID trong CSS.

    3. Vùng chọn dựa vào tên thẻ sẽ có cú pháp là tên thẻ
    Vùng chọn dựa vào ID sẽ có cú pháp là #ten-id
    Vùng chọn dựa vào Class sẽ có cú pháp là .ten-class

  3. So sánh 3 cách đặt CSS, cách nào là cách sử dụng phổ biến và hiệu quả nhất? Sinh viên đang sử dụng cách nào cho bài tập này?

    4. Cách 1 : Inline có ảnh hưởng ưu tiên cao nhất
    Cách 2 : Style tag có ảnh hưởng ưu tiên cao thứ hai
    Cách 3 : Đặt ở file CSS và nhúng vào qua file HTML qua thẻ link có ảnh hưởng ưu tiên thấp nhất
    Cách tạo file CSS riêng biệt được sử dụng phổ biến nhất vì có khả năng tái sử dụng, dễ chỉnh sửa
    Bài tập này được làm theo cách thứ 3.
